# Data Dictionary

## Dataset Overview

This dataset contains information about restaurants, including their location, services, ratings, and cost details.

## Column Details

1. Name: name
   Type: String
   Description: Name of the restaurant

---

2. Name: online_order
   Type: Integer (Binary)
   Description: Online ordering availability (1 = Yes, 0 = No)

---

3. Name: book_table
   Type: Integer (Binary)
   Description: Table booking availability (1 = Yes, 0 = No)

---

4. Name: rate
   Type: Float
   Description: Restaurant rating

---

5. Name: votes
   Type: Integer
   Description: Number of votes

---

6. Name: location
   Type: String
   Description: Restaurant location

---

7. Name: rest_type
   Type: String
   Description: Type of restaurant

---

8. Name: cuisines
   Type: String
   Description: Cuisine types

---

9. Name: approx_costfor_two_people
   Type: Integer
   Description: Approximate cost for two people

---

10. Name: listed_intype
    Type: String
    Description: Listing category

---

11. Name: listed_incity
    Type: String
    Description: City classification
